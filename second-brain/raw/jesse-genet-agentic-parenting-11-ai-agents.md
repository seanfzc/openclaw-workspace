# Jesse Genet: Agentic Parenting with 11 AI Agents

**Source**: a16z Podcast (April 13, 2026) 
**Jesse Genet**: ex-YC founder (Lumi), homeschooling mom of 4 kids under 5 
**Tags**: #ai #agents #parenting #homeschooling #obsidian #secondbrain #openclaw

---

## Executive Summary
Jesse runs **11 specialized AI agents** on local Mac Mini + Obsidian. 
Agents handle lesson plans, progress logs (voice + photos), groceries, Amazon orders, finances, toy inventory, and even code new agents. 
Goal: remove admin drudgery → more present parenting ("benevolent neglect").

---

## Core Workflow: Homeschooling Magic Loop
- Voice note + photos of books/materials → agent creates personalized lesson plan
- After session: 30-sec voice note + photos → agent writes beautiful Obsidian progress log
- Curriculum ingested from PDFs/books (e.g. *Building Foundations of Scientific Understanding*)
- Agents self-spawn new agents and maintain team docs

---

## Tech Stack
| Component | Details |
|---------------|--------------------------------------|
| Agents | 11 OpenClaw agents (most active) |
| Memory | Obsidian Markdown vault |
| Hardware | Dedicated Mac Mini (isolated profiles) |
| Models | OpenRouter (cost vs capability) |
| Key Feature | Self-spawning + autonomous updates |

- Agents have scoped permissions, own emails, strict "no impersonation" rules
- Runs entirely locally for privacy

## Household Automation Wins
- Grocery / Instacart orders
- Amazon shopping + birthday gifts
- Finance tracking
- Physical inventory (toys, supplies)
- Custom app coding (Jesse never used terminal before)

---

## Parenting Philosophy
- Structured 1:1 homeschool blocks + long unstructured play
- Builds kids' independence and boredom tolerance
- AI = tool for presence, not replacement
- Kids interact with named agents (e.g., "Sylvie" for curriculum) under supervision

---

## Risks & Values Alignment
- Agents are fed Jesse's chosen books and educational philosophy
- Voice interfaces tricky with young kids → testing e-ink tablets
- One incident: agent broke "no impersonation" rule → strict scoping now enforced
