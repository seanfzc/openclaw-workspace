# Instructions for AI Agents (Claude, Cursor, etc.)

## Task
Read everything in raw/. Compile a wiki in wiki/ following the rules in CLAUDE.md.

## Process
1. **Create INDEX.md first** — List every major topic you discover
2. **Create one .md file per major topic** in wiki/
3. **Each wiki file must**:
   - Start with a one-paragraph summary
   - Link related topics using [[topic-name]] format
4. **Summarize every source** — Extract key insights, patterns, and relationships
5. **Cross-link topics** — When themes overlap, create bidirectional links

## Rules
- Never modify files in raw/
- All new content goes into wiki/
- Keep summaries concise and factual
- Use consistent formatting across all files
