### Key Learnings

1. **Form vs Anchor handling is different:**
   - Forms need `e.preventDefault()` to handle via JavaScript
   - Anchor tags should NOT have `e.preventDefault()` for native navigation
   - Previous fix incorrectly removed preventDefault from form handler

2. **GitHub Actions permissions:**
   - `GITHUB_TOKEN` cannot push to repository by default
   - Auto-fix should be informational, not committing
   - Use `continue-on-error: true` for non-blocking analysis

3. **Test maintenance:**
   - Keep test selectors synchronized with HTML
   - Use semantic selectors (type attributes) when IDs missing
   - Regular validation against actual DOM structure
