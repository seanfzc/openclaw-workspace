## Executive Summary

The current ATOM-SG framework has significant gaps when compared to actual PSLE exam standards:

1. **Word Problems:** Current questions are linguistically simple and structurally basic. Real exams feature Cross-Thread Collision with 3-4 fused concepts, heavy supposition, and temporal misdirection.

2. **Geometry:** Current taxonomy covers only basic nano-nodes. Real exams feature Composite Overlap, Grid Construction, 3D Visualization, and Angle Chasing - none of which are in the current framework.

3. **Data Interpretation:** Currently missing as a formal pathway. Real exams feature Line Graphs with reverse percentage, rate of change analysis, and cumulative interpretation.

4. **Visual Quality:** Current renders lack exam-standard precision - no grid constructions, isometric 3D, composite overlaps, or reflex angle arcs.

### Exam Standard (ACS Junior Q16, Q17)

**Q16 - Erasers and Pencils (Cross-Thread Collision):**
```
Concepts fused:
1. Spending ratio (1/3 : 2/3)
2. Pack-size constraints (8 for $2, 5 for $3)
3. Quantity difference (36 more erasers)
4. Unit conversion (packs → items → money)

Linguistic complexity:
- "spent 1/3 of his money on erasers and the rest on pencils"
- "bought 36 more erasers than pencils"
- Pack images create visual-linguistic mismatch

Traps:
- Cross-Thread Collision (dominant)
- Unit Mismatch (packs vs items vs money)
- Implicit Constant (total money never restated)
- Visual-Linguistic Mismatch (Stress Score 3-4)
```

**Q17 - Orange Juice (Cross-Thread Collision):**
```
Concepts fused:
1. Sequential fractions (3/5, then 3/4 of remainder)
2. Shortfall/surplus adjustment (0.6ℓ short, 0.8ℓ left)
3. Simultaneous equations on same total
4. Nested fraction ambiguity

Linguistic complexity:
- "3/5 into 4 large bottles"
- "3/4 of the remainder into 9 small bottles"
- "wanted to fill 1 large bottle but was short by 0.6ℓ"
- "Instead, she filled 1 small bottle and had 0.8ℓ left"

Traps:
- Temporal Misdirection (multiple sequential states)
- Implicit Constant (total juice never stated)
- Nested Fraction Ambiguity
- Visual-Linguistic Mismatch (Stress Score ~4)
```

### Exam-Standard Geometry Pathways (NEW)

#### G5: Composite Overlap (NEW)
**Examples from ACS Junior:**
- Q15: Overlapping quarter circles with shaded lens region
- Q12: Five squares packed in rectangle with shaded leftover

**Required Visual Elements:**
- Overlapping shapes with intersection shading
- Area subtraction: (A + B - overlap)
- Perimeter of composite boundary
- Reflex angle arcs (>180°)

**Equation Shadows:**
```
Area_composite = Area_A + Area_B - Area_overlap
Perimeter_composite = External_boundary_only
```

#### G6: Grid Construction (NEW)
**Examples from ACS Junior:**
- Q9: Triangle on grid + complete trapezium BCDE
- Q3: Rectangle + rhombus composite (implied grid precision)

**Required Visual Elements:**
- Square grid with 1cm precision
- Construction tasks: "Complete the drawing"
- Multiple constraints: BC∥DE, AB=BE, DE=2×BC
- Protractor measurement on grid

**Equation Shadows:**
```
Constraint satisfaction system:
- Parallel lines (same slope)
- Equal lengths (same grid distance)
- Length ratios (2×, 3×, etc.)
```

#### G7: 3D Visualization (NEW)
**Examples from ACS Junior:**
- Q5: 8-cube solid with isometric view + orthographic projections
- Q19 (Nanyang): 7-cube solid, draw front view
- Q20 (Nanyang): Cuboid completion

**Required Visual Elements:**
- Isometric cube drawings
- Directional arrows (Top/Front/Side)
- Dot grids for construction
- Hidden vs visible edges

**Equation Shadows:**
```
Front_view_height = max(cubes_in_column)
Max_additions = hidden_positions_count
Cuboid_volume = l × b × h
```

#### G8: Angle Chasing (NEW)
**Examples from ACS Junior:**
- Q13: Rhombus ABCD + trapezium ADEF with AF∥DE
- Q3: Rectangle PQRS + rhombus RTUV with reflex angle 256°

**Required Visual Elements:**
- Multiple polygons sharing vertices
- Labeled angles (21°, 108°, 33°, etc.)
- Reflex angle arcs
- Parallel line indicators
- Intersection points (G, H)

**Equation Shadows:**
```
Rhombus: opposite angles equal, consecutive supplementary
Trapezium: consecutive interior angles supplementary
Around point: sum = 360°
Chain: ∠A → ∠B → ∠C (sequential deduction)
```

**Q10 - Line Graph (Data Interpretation):**

Parts:
(a) Day with most sold → requires interpreting drops in unsold line
(b) Percentage sold in first 5 days → cumulative calculation
(c) Original price before 20% discount → reverse percentage

Equation Shadows:
Daily_sold = unsold_day_n-1 - unsold_day_n
Cumulative_sold = 120 - unsold_day_5
Original_price = discounted_revenue / (0.8 × quantity_sold)

## Appendix: Example Exam-Quality Problem Specifications

### Example 1: Grid Construction (G6)
```yaml
problemID: G026
pathway: G6-Grid Construction
difficulty: Hard
visual_specs:
  grid_size: [10, 8]
  existing_shape:
    type: triangle
    vertices: {A: [1,1], B: [2,4], C: [6,2]}
  construction_task:
    type: trapezium
    name: BCDE
    constraints:
      - BC parallel to DE
      - AB equals BE
      - DE is twice BC
  answer_format: drawn_on_grid
equation_shadow:
  - Parallel: slope_BC = slope_DE
  - Equal_length: distance_AB = distance_BE
  - Ratio: length_DE = 2 × length_BC
traps:
  - Visual-Linguistic Mismatch: diagonal constraints on square grid
  - Implicit Constant: grid unit = 1cm never stated
  - Comparison Inversion: DE = 2×BC vs BC = ½×DE
```

### Example 2: Composite Overlap (G5)
```yaml
problemID: G027
pathway: G5-Composite Overlap
difficulty: Hard
visual_specs:
  shapes:
    - type: quarter_circle
      center: O
      radius: 10
      arc: A to C
    - type: quarter_circle
      center: O
      radius: 10
      arc: B to D
  overlap_region:
    name: OBC
    shading: diagonal_hatching
    given_area: 30
    given_perimeter: 26
  labels: [O, A, B, C, D]
parts:
  - Find area of figure OABCD
  - Find perimeter of figure OABCD
equation_shadow:
  area: 2 × (¼ × π × 10²) - 30 = 50π - 30
  perimeter: arc_AC + arc_BD + OA + OD
traps:
  - Visual-Linguistic Mismatch: shaded region is overlap, not single sector
  - Implicit Constant: radius 10 applies to all points OA=OB=OC=OD
  - Unit Mismatch: area in cm², perimeter in cm, π = 3.14
```
